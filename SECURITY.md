# Security Policy

## Scope
This repository contains early workflow patterns and design notes related to OpenClaw use in HR.

It does not contain production software, but security and privacy concerns still matter because the subject matter includes HR-sensitive workflows.

## Reporting
If you believe this repository contains:
- sensitive data that should not be public
- privacy-risky examples
- unsafe security guidance
- workflow patterns that could encourage inappropriate autonomous employment decisions

please report it privately to the repository owner instead of opening a public issue with sensitive details.

## Usage note
Nothing in this repository should be treated as approval to use real employee PII, confidential investigation data, or production HR records in demos, examples, or public workflow experiments.

## Current posture
This is an early public pattern repo, not a production implementation guide. Human review, privacy review, and legal review should remain in the loop for real-world HR use.
